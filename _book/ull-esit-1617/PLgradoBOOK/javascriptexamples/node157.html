Práctica: Calculadora con Listas de Expresiones y Variables

Modifique la calculadora vista en la sección anterior #section:calculadora#15417> para que el lenguaje cumpla los siguientes requisitos:
  • Extienda el lenguaje de la calculadora para que admita expresiones de asignación 2086
  • Extienda el lenguaje de la calculadora para que admita listas de sentencias 2087
  • El analizador devuelve la lista de expresiones evaluadas y la tabla de símbolos (con las parejas variable-valor).
  • Emita un mensaje de error específico si se intentan modificar las constantes 2088 y 2089.
  • Emita un mensaje de error específico si se intenta una división por cero
  • Emita un mensaje de error específico si se intenta acceder para lectura a una variable no inicializada 2090
  • El lenguaje debería admitir expresiones vacías, estos es secuencias consecutivas de puntos y comas sin producir error (2091)
  • Introduzca pruebas unitarias como las descritas en la sección #section:tstingfacil#15419> (<#15755#>Quick Tip: Quick and Easy JavaScript Testing with “Assert”<#15755#>)
